# Line Cracker

Display warning when a line of code is too long.

## Installation

``` sh
cargo add line_cracker --dev
```

Or add the following to your `Cargo.toml`:

``` toml
[dev-dependencies]
line_cracker = "0.1"
```

## Usage

``` sh
cargo line-cracker <fine_name>
```
